# shell-script
Comandos e estruturas do shell script

Comentário

```shell
# Meu comentário
```

Criaçāo de variáveis

```shell
x=0
```

Exibir o valor da variáveis no terminal

```shell
echo $x
```

Operação de Matemáticas

```shell
x=$(($x+1)) 
x=$(($x-1)) 
x=$(($x/2)) 
x=$(($x*1)) 
```

Operação entre duas variáveis

```shell
z=$(($x*$y))
```
